External Penetration Test Executive Summary Report Prepared for: Arclight Cloud Solutions, Inc. Testing Period: April 14-25, 2025 Report Date: May 5, 2025 Conducted by: NorthPoint Security Labs, an independent security testing firm Confidential - Shared under NDA for customer due diligence purposes
Executive Summary NorthPoint Security Labs performed an external penetration test of Arclight Cloud Solutions, Inc.'s production web application and supporting API infrastructure between April 14 and April 25, 2025. Testing followed a methodology aligned with the OWASP Testing Guide and NIST SP 800-115. SCOPE. - Production web application (app.arclightcloud.com) - Public API endpoints (api.arclightcloud.com) - Authentication and session management mechanisms SUMMARY OF FINDINGS. Testing identified 1 Medium-severity finding, 3 Low-severity findings, and 2 Informational observations. No Critical or High-severity vulnerabilities were identified during this engagement.
Summary of Findings ID Severity Finding Status AL-2025-01 Medium Verbose error messages disclosed internal Remediated stack traces on malformed API requests. AL-2025-02 Low Missing HTTP security headers (X-Frame-Options) Remediated on several static asset responses. AL-2025-03 Low Session cookies missing SameSite attribute on Remediated one legacy subdomain. AL-2025-04 Low Outdated TLS cipher suite supported for backward Remediated compatibility on one load balancer. AL-2025-05 Informational Rate limiting could be tuned more aggressively Accepted risk on password reset endpoint. AL-2025-06 Informational Recommend periodic review of third-party Acknowledged JavaScript dependencies loaded client-side. Remediation Status As of the report date, 4 of 4 actionable findings (AL-2025-01 through AL-2025-04) have been remediated and independently retested by NorthPoint Security Labs. The two informational observations do not represent vulnerabilities and are tracked as risk-accepted or acknowledged process improvements.
